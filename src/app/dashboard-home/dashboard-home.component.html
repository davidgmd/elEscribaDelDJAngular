Introducción

ElEscribaDelDJ es una aplicación rolera, creada para mi proyecto de fin de grado, su principal funcionalidad es ayudar con el tema de organizar fechas de partidas organizar resumenes de reglas, y tener todas las imagenes, documentos etc, organizados y listos para abrir mientras se realiza una partida ya sea presencial o telematicamente no esta pensada todavia para integrarla con roll20 u otras plataformas similares, si no más bien para partidas por discord u otros medios como skype etc.
